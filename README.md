# Adac Front-end Test

## Sobre o projeto

Este projeto foi desenvolvido como parte de um teste técnico para estágio, com o objetivo de reproduzir uma landing page moderna e responsiva a partir de um layout proposto.

A aplicação foi construída utilizando React + Vite, com foco em:

* Estruturação de componentes reutilizáveis

* Organização de layout

* Estilização moderna

* Responsividade (desktop e mobile)

* Boas práticas de front-end

## Tecnologias utilizadas:

* React — Biblioteca para construção da interface
* Vite — Ferramenta de build e ambiente de desenvolvimento
* CSS / Tailwind — Estilização da aplicação

## Funcionalidades implementadas:

* Layout fiel ao design proposto
* Estrutura em componentes React
* Seções organizadas (Hero, Features, Contents, Gallery, Partners, Testimonials, Footer)
* Botões com estilos e estados visuais
* Grid e Flexbox para organização dos elementos
* Responsividade para diferentes tamanhos de tela

## Decisões técnicas

* Utilização do Vite pela rapidez no ambiente de desenvolvimento

* Separação da interface em componentes para melhor manutenção

* Organização de estilos por seção para facilitar leitura

* Uso de classes semânticas e estrutura clara de layout

* Utilizando principalmente Flexbox, Grid e media queries.
